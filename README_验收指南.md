## 快速开始

### 1. 访问系统
```
http://localhost:3008
```

### 2. 测试步骤
1. 登录系统
2. 进入Chat页面（使用任意agent）
3. 查看右侧是否出现"最近文件"面板
4. 验证以下功能：
   - ✅ 显示非代码文件列表
   - ✅ 可以展开/收起面板
   - ✅ 可以下载文件
   - ✅ 代码文件被过滤

### 3. 实时更新测试
在Chat中输入：
```
帮我创建一个名为测试文件.txt的文件，内容是'测试实时更新'
```
等待5秒，查看面板是否自动刷新显示新文件。
